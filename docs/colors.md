# colors.py 

This module contains the RGB color tuple definitions.

Format: `(red, green, blue)`

# Variables

## Primary Colors
- `RED`
- `GREEN` 
- `BLUE` 

## CMYK Colors
- `CYAN`
- `MAGENTA`
- `YELLOW`
- `BLACK`

## Grayscale
- `WHITE`
- `GRAY`
- `SILVER`

## Mixed
- `PURPLE`
- `TEAL`
- `DARK_RED`
- `DARK_YELLOW`
- `DARK_GREEN`
- `DARK_BLUE`
